# lowCode-editor

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```

### Git commit message 

+ feat - 新功能 feature
+ fix - 修复 bug
+ docs - 文档注释
+ style - 代码格式(不影响代码运行的变动)
+ refactor - 重构、优化(既不增加新功能，也不是修复bug)
+ perf - 性能优化
+ test - 增加测试
+ chore - 构建过程或辅助工具的变动
+ revert - 回退
+ build - 打包
